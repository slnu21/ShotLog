## 한국어

### 짧은 설명 (Short description)
화면을 캡처하고 곧바로 메모를 붙여, 프리셋 폴더에 정리하고 Markdown으로 내보내는 트레이 도구.

### 설명 (Description)
ShotLog는 "캡처하고, 메모하고, 한 번에 정리"를 위한 가벼운 시스템 트레이 앱입니다. 단축키 한 번으로 화면을 잡고, 떠오른 생각을 바로 적어 두면, 나중에 인박스에서 검토하고 Markdown 문서로 깔끔하게 내보낼 수 있습니다. 모든 데이터는 내 PC에만 저장되며, 네트워크 전송이 전혀 없습니다.

**주요 기능**
- 네 가지 캡처: 즉시 캡처(활성 모니터 → 활성 프리셋 바로 저장), 캡처+메모, 영역 선택, 활성 창
- 전역 단축키 — 어떤 앱 위에서도 동작, 단축키는 직접 변경 가능
- 프리셋 — 저장 폴더·색상·기본 태그를 묶어 두고 캡처 시 한 번에 선택
- 인박스 — 최근 캡처를 썸네일로 모아 보고, 메모/태그를 즉석에서 편집·검색·필터·삭제
- 글쓰기 내보내기 — 태그/프리셋/기간으로 골라 시간순 Markdown 문서로 내보내기(이미지 폴더 포함, GitHub·Obsidian·VS Code에서 그대로 열림)
- 사이드카 — PNG 옆에 같은 이름의 .md를 함께 저장해 탐색기에서도 메모 확인(선택)
- 트레이 상주 · 단일 인스턴스 · Windows 시작 시 자동 실행(선택)
- 로컬 전용 — 스크린샷과 메모는 내 기기에만 저장, 외부 전송·추적 없음

## English

### Short description
A tray app to capture your screen, jot a note on the spot, sort it into preset folders, and export to Markdown.

### Description
ShotLog is a lightweight system-tray app for "capture, note, organize — in one go." Grab the screen with a single hotkey, type the thought that prompted it, then review everything later in the Inbox and export a clean Markdown document. Everything stays on your PC — ShotLog makes no network calls.

**Highlights**
- Four capture modes: instant (active monitor → active preset), capture + note, region select, and active window
- Global hotkeys that work over any app, fully remappable
- Presets that bundle a destination folder, an accent color, and default tags — pick one at capture time
- Inbox to browse recent captures as thumbnails and edit memos/tags inline, with search, preset filter, and delete
- Export to portable Markdown — filter by tag/preset/date, pick what to include, and get a chronological document plus an images folder that opens anywhere (GitHub, Obsidian, VS Code)
- Optional sidecar: a same-named .md next to each PNG so notes are visible in Explorer too
- Tray-resident, single-instance, optional start-with-Windows
- Local-only: screenshots and notes are stored on your device — no uploads, no tracking

### What's new — v1.0.0
Initial release: four capture modes, presets, Inbox, Markdown export, sidecars, and a refreshed unified icon.
